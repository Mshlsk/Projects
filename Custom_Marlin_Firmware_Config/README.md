# 🛠 Custom Marlin Firmware Configuration (MKS Base)

## 📝 О проекте
Здесь представлена кастомизированная конфигурация прошивки Marlin для домашнего 3D-принтера с дельта-кинематикой. В этом репозитории лежат настроенные файлы `Configuration.h` и `Configuration_adv.h`, которые я адаптировал под конкретное железо. 

Этот проект демонстрирует мои навыки работы с embedded-системами: чтение документации, конфигурация аппаратных интерфейсов, калибровка сложной кинематики и настройка систем управления.

## ⚙️ Hardware Stack & Kinematics
* **Кинематика:** Delta (сложный расчет траекторий, одновременная работа 3 осей для перемещения).
* **Управляющая плата:** MKS Base.
* **Драйверы:** A4988.
* **Управление и дебаг:** Pronterface (отправка G-кодов, чтение логов термисторов).

## 🚀 Реализованные фичи и настройки
* **Delta Kinematics Calibration:** Настроены геометрические параметры дельты (DELTA_PRINTABLE_RADIUS 110.0, DELTA_HEIGHT 371.96, длина тяг).
* **Auto Bed Leveling (Bilinear):** Настроено автоматическое выравнивание стола по сетке с учетом оффсетов щупа (NOZZLE_TO_PROBE_OFFSET { 0, 0, 1.10 }).
* **Thermal Runaway Protection:** Активирована программная защита от перегрева и возгорания для экструдера и нагревательного стола.
* **PID Tuning:** Включен и настроен PID-регулятор (PIDTEMP) для максимально стабильного удержания температуры хотэнда без перерегулирования.
* **Preheat Presets:** Заданы кастомные пресеты для быстрого прогрева под филаменты PLA (180°C / 70°C) и ABS (240°C / 110°C).

## 📂 Структура репозитория
* `Configuration.h` — Основной файл с настройками назначения пинов, кинематики и термоконтроля.
* `Configuration_adv.h` — Расширенные настройки прошивки.
